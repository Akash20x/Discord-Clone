## 📝 Technologies
Discord Clone is built using the following stack & some small libraries:

### Frontend
- ReactJS
- Moment
- Pusher
- React Modal
- React Router

### Backend
- ExpressJs
- Mongoose
- Jsonwebtoken
- Bcryptjs
- CORS
- Pusher
- Deploy on Heroku

## 💡 Features
- User Authentication using Login and Register
- Create channels
- Send messages in channels
- Browse existing channels
- Mobile Responsive
